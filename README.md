# Java Practice Collection

A personal collection of Java practice programs covering core Java concepts, collections, exception handling, object-oriented programming, and common coding problems.

## Project Structure

- `Collection_Framework/` - Java collection examples and demos.
  - `ArrayList/` - Add, iterate, and remove elements from `ArrayList`.
  - `LinkedList/` - LinkedList usage and methods.
  - `Queue/`, `PriorityQueue/`, `Stack/`, `Vector/`, `TreeSet/`, `LinkedHashSet/` - examples for queue, priority queue, stack, vector, tree set, and linked hash set.
  - `Map/` - Map creation and update operations.
  - `ArrauDqueue/` - ArrayDeque examples.

- `ExceptionHandling/` - Java exception handling demos.
  - `CheckedUnchecked.java` - Difference between checked and unchecked exceptions.
  - `TryCatchImplementation.java`, `TryWithoutCatch.java`, `TryWithResources.java` - try/catch examples and resource management.
  - `ThrowVsThrows.java`, `CustomExceptionDemo.java`, `InvalidAgeException.java` - custom exceptions and exception propagation.
  - `StackOverflowDemo.java`, `DevideFromArrayException.java` - error demos and exception handling.

- `java problem question/` - Common Java problem solutions.
  - `avg.java` - average calculation.
  - `EligibilituForVoting.java` - voting eligibility.
  - `FindGretest.java` - find greatest number.
  - `LeepYear.java` - leap year checking.
  - `oddnumbers.java` - odd number logic.
  - `pattern1.java`, `PositiveAndNegative.java`, `Table.java` - pattern printing and number classification.

- `OPPS/` - Object-oriented programming examples.
  - `Abstraction/` - abstract classes and shapes.
  - `Classes_Objects/` - class and object examples like `Car` and `Student`.
  - `Constructors/` - constructor examples.
  - `Encapsulation/` - encapsulation demo with `Human`.
  - `Inheritance/` - inheritance examples with `Animal`, `Person`, `Vehicle`, and related classes.

- `Opps_Practice/` - Additional OOP practice files.
  - `Encapsulation/BankingQuestion/` - banking question practice.
  - `dfs/` - placeholder for DFS practice.

- `Pattern_Programs/` - pattern printing examples.
  - `pattern1.java`, `pattern2.java`, `pattern3.java`, `pattern4.java`, `pattern5.java`

## Getting Started

1. Open the folder in your Java IDE (Eclipse, IntelliJ IDEA, VS Code).
2. Locate the desired `.java` file in the folder structure.
3. Run the file using your IDE or compile from the command line:

   ```bash
   javac FileName.java
   java FileName
   ```

   Example:
   ```bash
   cd "c:\\Users\\visha\\Desktop\\Java practice\\java problem question"
   javac EligibilituForVoting.java
   java EligibilituForVoting
   ```

## Notes

- Many files are simple Java practice exercises and may use package-free classes.
- For collection and OOP examples, run the class with the `main` method inside each file.
- Some directories are named with misspellings (`ArrauDqueue`, `LeepYear`, `EligibilituForVoting`) but still contain usable Java examples.

## Recommended Learning Path

1. Start with `java problem question/` to practice basic logic and control flow.
2. Move to `Pattern_Programs/` for loops and output formatting.
3. Explore `OPPS/` for object-oriented concepts.
4. Study `Collection_Framework/` for Java collection utilities.
5. Review `ExceptionHandling/` to understand Java error handling.
